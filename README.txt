CarbonProject_Final
===================

📘 專案簡介：
這是一個由 Jennie 改良整合的 ASP.NET MVC 專案，
結合組員原版 CarbonProject 後端邏輯與 CustosCarbon 粉白前端風格。

🌸 特色：
- MySQL 資料庫連線（跨平台支援）
- Session 登入狀態管理
- BCrypt 密碼加密
- CustosCarbon 粉白 UI 設計
- 含 Privacy / Refrences 原文保留版

⚙️ 執行步驟：
1️⃣ 啟動 MySQL：
    brew services start mysql
2️⃣ 登入 MySQL：
    mysql -u root -p
3️⃣ 建立資料庫：
    CREATE DATABASE carbon;
4️⃣ 執行專案：
    dotnet run
5️⃣ 瀏覽：
    https://localhost:7157/
    （首頁會顯示粉白登入介面）

✅ 適用環境：
- macOS / Windows
- .NET SDK 8.0 以上
- MySQL 8.0 以上
